# Images Folder

## How to Add the Founder Image

1. **Save the founder image** (the one you provided) to this folder as:
   ```
   public/images/alex-founder.jpg
   ```

2. The image is already configured in `components/AboutSection.tsx` to use this path.

3. Make sure the image is:
   - Portrait orientation (4:5 aspect ratio works best)
   - High quality (at least 800x1000px)
   - JPG or PNG format

## Other Images to Add

You can also add these images to enhance the site:

- `hero-bg.jpg` - Hero section background
- `selection-prep.jpg` - Selection Prep program card
- `tactical-hypertrophy.jpg` - Tactical Hypertrophy program card
- `dad-bod.jpg` - Dad Bod program card

All images will be automatically loaded from this folder.

## Key Design Elements

### 1. Typography
- **Headings:** 6xl to 9xl (96px to 128px)
- **All caps** with wide letter spacing
- **Bebas Neue** font for maximum impact
- **Tracking-widest** for aggressive look

### 2. Colors
- **Tactical Black:** #0A0A0A (primary background)
- **Tactical Red:** #C41E3A (accents, CTAs, borders)
- **White:** Pure white for contrast
- **No gradients** - solid colors only

### 3. Borders & Lines
- **2-4px solid borders** everywhere
- **Red accent lines** at section breaks
- **Box shadows** with offset borders
- **Sharp corners** (no border-radius)

### 4. Animations
- **Scale transforms** on hover (1.05-1.1x)
- **Slide-in effects** for sections
- **Fade-ins** with delays
- **Smooth 300-800ms transitions**

### 5. Layout
- **Full-screen hero** (h-screen)
- **Large padding** (py-32 = 128px)
- **Breathing room** between sections
- **Grid layouts** for programs and stats

## Typography Scale

```
text-9xl = 128px (Hero headline)
text-8xl = 96px  (Section headings)
text-7xl = 72px  (Stats numbers)
text-6xl = 60px  (Page titles)
text-5xl = 48px  (Subheadings)
text-4xl = 36px  (Card titles)
text-3xl = 30px  (Component headings)
```

## Spacing Scale

```
py-32 = 128px (Section padding)
py-24 = 96px  (Stats section)
py-20 = 80px  (Navigation height)
py-16 = 64px  (Footer padding)
gap-16 = 64px (Grid gaps)
```

## Animation Timings

```
duration-300 = 300ms (Quick hovers)
duration-700 = 700ms (Image scales)
duration-800 = 800ms (Section fades)
```
